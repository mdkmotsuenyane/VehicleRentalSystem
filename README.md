🚗 Vehicle Rental System

A simple Java-based Vehicle Rental System demonstrating core Object-Oriented Programming (OOP) concepts such as inheritance, encapsulation, and polymorphism.

📌 Project Overview

This console application allows users to:

View available vehicles
Rent a vehicle
Return a vehicle

The system supports multiple vehicle types, including:

🚘 Cars
🏍️ Motorcycles
🧠 OOP Concepts Used
Encapsulation → Private attributes with public getters
Inheritance → Car and Motorcycle extend Vehicle
Polymorphism → Vehicle array stores different object types
Method Overriding → Custom toString() in subclasses
🏗️ Project Structure
vehiclerentalsystem/
│
├── Vehicle.java        // Superclass
├── Car.java            // Subclass (Car)
├── Motorcycle.java     // Subclass (Motorcycle)
├── VehicleRentalSystem.java  // Main class (menu system)
⚙️ Features
📋 View all vehicles with details
🔑 Rent a vehicle using ID
🔄 Return a rented vehicle
✅ Availability tracking
💰 Price per day display
